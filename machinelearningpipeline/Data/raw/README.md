# Raw Data (synthetic)

This folder contains synthetic data used for development and testing.

Files
- `pima_diabetes_synthetic.csv` — Synthetic Pima Indians Diabetes dataset (768 rows).
- `generate_pima_synthetic.py` — Script to generate the CSV. Run with `python generate_pima_synthetic.py`.

Columns

- `Pregnancies`: Number of times pregnant
- `Glucose`: Plasma glucose concentration a 2 hours in an oral glucose tolerance test
- `BloodPressure`: Diastolic blood pressure (mm Hg)
- `SkinThickness`: Triceps skin fold thickness (mm)
- `Insulin`: 2-Hour serum insulin (mu U/ml)
- `BMI`: Body mass index (weight in kg/(height in m)^2)
- `DiabetesPedigreeFunction`: Diabetes pedigree function
- `Age`: Age (years)
- `Outcome`: Class variable (0: non-diabetic, 1: diabetic)

Note: The CSV is synthetic and not intended for any clinical use.
